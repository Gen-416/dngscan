# dngscan Assets

Bundled third-party assets used or referenced by dngscan.

- `tony_mc_mapface.spi3d` — Tony McMapface LUT used by `--jpeg-mode tony`, redistributed under the upstream MIT license in `TONY_LICENSE-MIT.md`.
- `darktable_agx.c` / `darktable_agx.cl` — reference copies from darktable's AgX implementation, redistributed under GPL-3.0-or-later.

The CLI still accepts `--tony-lut PATH` when you want to point to a different LUT.
