# dngscan_assets

Put local, non-committed runtime assets here.

- `tony_mc_mapface.spi3d` — optional Tony McMapface LUT used by `--jpeg-mode tony`.
- `darktable_agx.c` / `darktable_agx.cl` — local reference copies for AgX behavior.

The CLI still accepts `--tony-lut PATH` when you want to point to a different LUT.
